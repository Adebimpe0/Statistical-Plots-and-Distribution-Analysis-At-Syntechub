# Statistical Plots and Distribution Analysis

This project performs **exploratory statistical analysis**, **distribution visualization**, and **outlier detection** on the Superstore dataset.  
It includes histograms, KDE plots, boxplots, category-wise distribution analysis, and IQR-based outlier detection.
